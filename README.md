# React Portfolio
###
This react app is using Google search book api along with Mongodb and allow user to search, save and remove book from booklist.


# Techiniques:
###
<ul>
<li>React</li>
<li>JSX</li>
<li>Javascript</li>
<li>React Route</li>
<li>MongoDB</li>
<li>Mongoose</li>
</ul>
